## Features

- **Anti-Bot Resilience:** Stealth-configured Selenium to bypass advanced bot detection.
- **Smart RNW Parsing:** Layout-agnostic extraction for "React Native Web" structures.
- **Glassmorphism UI:** Apple-inspired dashboard with frosted glass effects and dynamic star ratings.
- **Performance:** Parallelized-ready logic with optimized navigation pipelines.
- **One-Click Deploy:** Fully containerized with Docker support.

## Quick Start

### 1. Prerequisites
- Python 3.9+
- Google Chrome installed.

### 2. Installation
```bash
git clone https://github.com/YashI2IT/Bharat-Soft-Solutions-Internship-Task-1.git
cd Bharat-Soft-Solutions-Internship-Task-1
pip install -r requirements.txt
```

### 3. Run Locally
```bash
python app.py
```
Open [http://localhost:5000](http://localhost:5000).

---

## Deployment via Docker

```bash
docker build -t flipkart-scraper .
docker run -p 5000:5000 flipkart-scraper
```
